O(n²)

🎯
# 06
Selection Sort
Mencari elemen minimum/maksimum dan menempatkannya pada posisi yang benar secara berulang.
O(n²)
→
📥
# 07
Insertion Sort
Membangun array terurut satu per satu dengan menyisipkan elemen ke posisi yang tepat.
O(n²)
→

# 08
Merge Sort
Divide and conquer: membagi array menjadi dua bagian, sort, lalu gabungkan kembali.
O(n log n)

🚀

Quick Sort
Algoritma sorting tercepat rata-rata. Memilih pivot dan mempartisi array secara rekursif.
O(n log n)
